📖 第一章：初识CSS

什么是CSS？主要作用是做啥的？

CSS（Cascading Style Sheets，层叠样式表）是用于控制网页视觉表现的语言。如果说HTML是网页的骨架，那么CSS就是网页的皮肤和衣服。

CSS根据书写位置，可以分为几类？

根据书写位置，可以分为以下三类：

  • 行内样式表-写到标签内部可以控制，当前标签的样式。
  • 内部样式表-写到head标签中，可以控制当前页面的样式。
  • 外部样式表-独新建一个CSS文件，控制整个网站的样式。

🎨 第四章：字体样式 - 文字的”美容师”

字体颜色 color

  • 1.关键字，例如red、grenn、blue等
  • 2.十六进制，十六进制颜色值由6个十六进制数字组成，每个数字的范围是0-9、a-f
  • 3.rgb/rgba格式，rgba由红色、绿色、蓝色三个通道组成，每个通道的取值范围是0-255，透明度通道的取值范围是0-1
  • 4.hsl/hsla格式，hsl颜色值由色相、饱和度和亮度组成，每个值的范围是0-360、0-100%、0-100%
    📚四种颜色表示方式对比
    表示方式 语法示例 使用场景 使用频率 优点
    关键字 color: pink; 学习使用 ⭐⭐ 简单直观，易记忆
    十六进制 color: #ff0000; 开发最常用 ⭐⭐⭐⭐⭐ 精确控制，兼容性好
    rgb/rgba color: rgba(49, 233, 64, 0.1); 动态颜色、透明度 ⭐⭐ 直观，可计算
    hsl/hsla color: hsl(0, 100%, 50%); 动态主题、颜色计算、设计系统 ⭐⭐⭐ 最符合直觉，易于调整，语义化强

📚 第八章：CSS三大特性-优先级

  • 1.继承性
  • 2.重叠性
  • 3.优先级
    继承性和重叠性前面已提过，这里不再赘述。
    优先级原则：
  • 1.优先级相等的时候，CSS中最后的那个声明的样式会被应用到元素上。（层叠遵循就近原则）
  • 2.其余判断哪个选择器的权重高，优先执行那个样式。
  • 3.权重是4位一组，是分开的层级，不能进位。
    选择器类型 示例 权重值 优先级说明
    !important color:red !important 无限大 强制覆盖所有规则
    内联样式
    		(1,0,0,0) 行内样式权重最高(1,0,0,0)
    ID选择器 #myid (0,1,0,0) 每个ID增加0,1,0,0
    类/属性/伪类 .class，[type=”text”] (0,0,1,0) 每个类/属性/伪类增加0,0,1,0
    类型（标签）/伪元素 div，::before (0,0,0,1) 每个标签增加0,0,0,1
    通配符/继承 *，继承的样式 (0,0,0,0) 通配符和继承性权重最低

📌 第九章：盒子模型

所有的元素都被一个个的“盒子”包围着，学会盒子模型可以实现精准布局、处理元素排列的关键。
在CSS中，我们有几种类型的盒子，一般区分为块级盒子(block boxes)和行内盒子(inline boxes)
块级盒子(block):

  • 1.盒子会产生换行。
  • 2.width和height属性可以发挥作用
  • 3.不设置宽度，则默认是父元素空间的100%
  • 4.内边距、外边距会撑大元素
  • 5.常见的比如：div、p、h、ul、table等
    行内盒子(inline):
  • 1.盒子不会产生换行
  • 2.width和height属性将不起作用
  • 3.垂直方向的内边距、外边距不起效果
  • 4.水平方向的内边距、外边距会有效果
  • 5.常见的比如:span、em、a、strong等
    盒子模型的组成：
    CSS盒模型整体上适用于区块盒子，包含四部分:
  • 1.盒子内容，显示内容的区域，由内容或指定宽度高度来决定内容的大小。
  • 2.内边距padding，内容距离边框之间的距离。
  • 3.外边距border，该盒子与其他元素之间的距离。
  • 4.边框margin，边框盒子包住内容和内边距。
