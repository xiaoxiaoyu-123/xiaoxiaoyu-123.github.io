▶️ 运行项目

启动开发服务器：

  • 服务器启动后，在浏览器中访问终端显示的地址（通常是 http://localhost:5173）
  • 修改代码时，页面会自动刷新

💡 实用提示

  • 依赖问题：删除 node_modules 和 pnpm-lock.yaml 后重新运行 pnpm install
  • 端口占用：在 package.json 中修改端口配置
  • 构建项目：使用 pnpm build 命令构建生产版本

🎯 总结

核心步骤：

  1. 安装 Node.js
  2. 安装 pnpm
  3. 安装项目依赖：pnpm install
  4. 运行项目：pnpm dev
